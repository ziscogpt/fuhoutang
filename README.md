# 拙诚版 · 富厚堂

《曾国藩全集》专题数据集在故居现场的阅读器——按《拙诚版·富厚堂小程序.html》设计方案实现的移动端 Web 原型。

## 在线体验

https://ziscogpt.github.io/fuhoutang/ (手机浏览器打开体验最佳)

## 本地运行

```
node server.js          # 默认 8000 端口,可加参数换端口: node server.js 9000
```

启动后:

- 电脑预览:打开 http://localhost:8000 (浏览器开发者工具切成手机模拟器体验最佳)
- **手机体验:手机连到和电脑同一个 Wi-Fi,浏览器/微信里打开 `http://<电脑局域网IP>:8000`**
  (服务启动时会把地址打印在终端里;若打不开,放行 Windows 防火墙对 Node 的入站规则,或
  `netsh advfirewall firewall add rule name="fht8000" dir=in action=allow protocol=TCP localport=8000`)

无需安装任何依赖,纯静态文件 + Node 内置 http 模块。

## 体验路线(照着产品的一天走)

1. **门口扫码** → 落地页,半个印落下(红:你来过) → 进园宅图
2. **舆图**:点七处任一点位进内容页(笨事实 · 听讲解 · "我到了这一处"落一笔,右上"拙"字长一笔);门房是第八处,原声不剪辑
3. **四格**:说件小事(写半封信,可代寄 ¥38)· 问一问(句句带记录号,点记录号下钻出处页)· 试一天(日课四条,做完自己圈)· 任务卡(给孩子)
4. **问先生**常驻底部导航(顶层四页:舆图/首页 · 问先生 · 书房 · 我的;深层页不出导航,保持沉浸)——他只用自己说过的话回答,没写过的直说没有;问一问与问先生可互相切换
5. **离园清单**(舆图顶部)→ 走满八处才有**字卡**;点"我离园了"切到行后状态
6. **行后首页**:百日课(断了不清零)· 接着读(电子书)· 对印(两半合一,再寄一封)· 铺子 · 书房(守宅人 / 七次 / 每月读半封)
7. **我的**:半印卡、我的信、摘录、订阅、订单;页底有"演示控制"(在园/离园切换、时间快进、八处走满、重置)

## 结构

```
docs/             站点本体(GitHub Pages 从这里发布)
  index.html      壳(字体、toast、支付半屏)
  css/app.css     设计规范:方不圆 / 格不飘 / 红是用户挣的
  js/data.js      内容数据(模拟专题数据集,逐条带记录号)
  js/app.js       状态机 + 哈希路由 + 全部页面视图
server.js         零依赖静态服务器(本地预览用)
smoke-test.js     冒烟测试:node smoke-test.js(渲染全部路由两种状态)
```

状态存在浏览器 localStorage(`fht_state_v1`),对应设计里"不用注册,跟着你的微信走"。
